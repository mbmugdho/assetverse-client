# AssetVerse (Client)

A B2B HR & Asset Management frontend built with React, Vite, and DaisyUI.

## Live URL
https://assetverse-client-five.vercel.app/
## Purpose
AssetVerse helps companies manage physical assets and track which employee has which equipment.

## Key Features
- HR & Employee roles with Firebase authentication
- Asset inventory management
- Asset request & approval workflow
- Auto-affiliation & multi-company employees
- HR analytics with Recharts
- Stripe-powered package upgrades

## Tech Stack / Main npm Packages
- React, Vite, React Router
- @tanstack/react-query
- Axios
- Firebase
- DaisyUI / Tailwind CSS
- Framer Motion
- Recharts
- SweetAlert2
- Swiper
